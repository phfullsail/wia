Adipoli

What is Adipoli?

Adipoli is a jquery plugin for creating image hover effects.

There are more than 15 different hover effects that can be applied to images.

In addition to hover effects there are many other image styles available in this plugin.

Why Adipoli?

  1. Impressive image hover effects.
  2. Easy to implement.

What is needed to get started?

Download Adipoli

https://cube3x.com/adipoli-jquery-image-hover-plugin/

Link to adipoli css in the head

<link href="css/adipoli.css" rel="stylesheet" type="text/css"/>

Include the lastest jquery script in the head tag

<script src="includes/jquery-1.9.1.js" type="text/javascript"></script>

Include adipoli script in the head tag

<script src="js/jquery.adipoli.min.js" type="text/javascript"></script>

Start Effects

  3. overlay

Some of the Hover Effects
